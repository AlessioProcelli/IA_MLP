
# IA_MLP
## Dipendenze
Per far funzionare correttamente il programma si necessita del dataset Zalando che come richiesto non è inserito all'interno. si può scaricare da questo link https://github.com/zalandoresearch/fashion-mnist.
## Struttura del progetto
Cè  un solo file main.py in cui è inserito il progetto composto da due funzio di  funzionamento (train,prediction)  una funzione di inizzializzazione (inizializza_dati) e due funzioni di supporto alle funzioni principali.  in fondo al file si trova il codice che viene messo in esecuzione per simulare quanto richiesto nel progetto.
